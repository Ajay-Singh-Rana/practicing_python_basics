The help() and dir() Functions

Today, we are gonna learn about two most useful functions in Python...the help() and the dir() functions.Let's start with help() function first.

What does the help() function provide us with?

  • This is a wrapper around pydoc.help that provides a helpful message when 'help' is typed at the Python interactive prompt.
  • Calling help() at the Python prompt starts an interactive help session.
  • Calling help(thing) prints help for the python object 'thing'.

Enough of those heavy words...In Simple words:
The help() function is a function that provides us with some basic documentation (docstrings) on Python objects,modules,functions and classes. This documentation helps us know about how to use that object or module or function or what functionality that class provides us with. Let's see it in action with the time module of Python.

Note: Don't use the parenthesis () when taking help on a function, just pass its name.

import time # importing time
print(help(time)) # using the help() function to access the documentation on the time module

All that which appears above on the screen is the documentation on the time module and is pulled out by parsing the doctrings of that module. Now, let us see what the dir() function has to offer.

The dir() function

The dir() function according to it's documentation pulled out using the help() function :

  • for a module object: the module's attributes.
  • for a class object: its attributes, and recursively the attributes of its bases.
  • for any other object: its attributes, its class's attributes, and recursively the attributes of its class's base classes.

Enough of those heavy words...Simply, the dir() function lists all the attributes,constants,functions and classes that are available to use with the module/class/object. Let's now see it in action...we'll use it on the time module.

dir(time) # lists all the functions,attributes and classes available for use

All those names in all small-caps are functions. So, let's use the help() function on one of these functions to see what it does. Let's use it on the time function.

help(time.time) 
Help on built-in function time in module time:

time(...)
    time() -> floating point number
    
    Return the current time in seconds since the Epoch.
    Fractions of a second may be present if the system clock provides
    them.

So, the help() says that to the time.time function returns the current time in seconds since the Epoch. So, let's check it out.

time.time()
1602674170.0587213

Whoa..! it does what the help() says. This is enough for demonstrating the power of the help() and the dir() functions.
We can learn about any module/function/class/object just by using these. Also, this shows us how important the doctstrings are.
These are the ones which get parsed to pop all this help on our screens.So, it is always good to use doctrings in our classes/modules/functions.
Cause these might help many other people to learn about our functions/classes/modules. Also, these are a great reminder to self.

Tkinter Window and the Label widget

Creating a Tkinter window

import tkinter as tk # importing the tkinter library with the alias name tk

window = tk.Tk() # calling the Tk() method creates a window
window.title("Our First Window") # setting the title of the window
window.geometry("350x300") # a string of "widthxheight" is passed to tell the size of the window

window.mainloop() # the mainloop() method listens for events until the widnow is closed

Firstly, to use the tkinter library we need to import it. The line import tkinter as tk does the importing task.

Now, to create a window, the tkinter library provides a method Tk().The line window = tk.Tk() calls the mehthod Tk() to create a window.

Fianlly, the line of code window.mainloop() listens for window events (such as button clicks or a keypress) until the window is not closed.

Label Widget

The Label widget is a text and image container in tkinter. It is used to add text/images to the main window. Though the text can be displayed in a single font but the text can span multiple lines.

The Label(master,**options) method is used to create labels. Here, master is the name of the root window where the label is to be added.

import tkinter as tk # importing the tkinter library
window = tk.Tk() # creating the tkinter window

textLabel = tk.Label(window,text="Hey There!",fg="red") # creating a Label
textLabel.pack() # adding the label to the main window with the help of pack() geomemtry manager

window.mainloop() # running the mainloop

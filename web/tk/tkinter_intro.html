Tkinter

Introduction

Tkinter is a GUI (Graphical User Interface) toolkit provided with standard Python installations. It is a Python binding for the Tk GUI toolkit
for the Tcl language. The name Tkinter comes from Tk Interfaces. Tkinter was written by Fredrick Lundh.It is used to create GUI applications in Python with the help of an embedded Tcl interpreter.

Other GUI toolkits for Python are PyQt,PySide,wxPython,Kivy and PyGTK.

Objectives

We have the following two objectives from this course :

1. Learn about Tkinter widgets.

2. Make a basic To-Do list with Python using Tkinter for a practical experience of using Tkinter.

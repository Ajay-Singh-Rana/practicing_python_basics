Pandas is a library for python to analyze the data in a simple and efficient way.

The read_csv method

The read_csv() method is used to read the contents of a csv file and it returns a DataFrame object which is easy to access. Indexes can be used to access data from the DataFrame data structure. A few methods that we can use to play with indices are : 1. iloc() # uses the integer location for the data items in the DataFrame. 2. loc() # uses the labels as location for accessing the data
